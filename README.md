若你想修正有关错别字和代码的问题，请不吝提交pull request。

若您希望自行编译HTML，安装Pandoc并执行``make``

```bash
$ make
```

使用下面的命令可以生成pdf和epub文档：

```bash
$ make tutorial.pdf
$ make tutorial.epub
```

执行下列命令相当于``make``并进行预览：

```bash
$ make preview
```

对于翻译者，执行下列命令进行发布：

```bash
$ make pub
```
